Document Object model

In JavaScript, the document object represents the current web page loaded in the browser. It serves as an entry point to access and manipulate the HTML content of the page.

console.dir()

console.dir() is a JavaScript method that displays an interactive list of an object's properties in the browser's console. It's particularly useful for inspecting complex objects like DOM elements and nested objects.
